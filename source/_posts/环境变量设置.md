## 系统版本

ubuntu22.04

## 操作命令

#### 1.对当前终端有效

```
echo 'export PATH=$PATH:~/.npm-global/bin' >> ~/.profile

source ~/.profile
```

#### 2.对当前用户有效

```
echo 'export PATH=$PATH:~/.npm-global/bin' >> ~/.bashrc

source ~/.bashrc
```

#### 3.对所有用户有效

```
echo 'export PATH=$PATH:~/.npm-global/bin' >> /etc/profile

source /etc/profile
```
